# 開發環境

!!! abstract "目標"

    - 隨手寫一個 App
    - 在電腦上執行
    - 在手機上執行

``` mermaid
graph LR
  A[畫面編排] --> B[程式設計];
  B --> C[驗證執行];
```

重點：

- 登入 [MIT App Inventor](https://appinventor.mit.edu/)
- **Create Apps!**
- 環境介紹
  - 畫面編排
  - 程式設計
- 連線
  - AI companion 程式
    - 用手機，需安裝 MIT AI2 Companion
    - 掃瞄 QR Code
  - 模擬器
    - 打包，用模擬器 (BlueStack、NOX) 執行
  - USB
